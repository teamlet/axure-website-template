# 项目简介

项目创建的目的是为了学习和使用 Axure。

在使用 Axure 过程中，搜集整理常用的 Axure 组件库。

最终使用这些组件库制作一些常见的网站模板。

项目使用 pyCharm 进行管理，使用了python mkdocs 进行文档编写。

## 项目计划

* 初始化项目。
* 上传 Axure 组件库。
* 制作网站模板。
